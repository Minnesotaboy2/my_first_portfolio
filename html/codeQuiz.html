This is currently a demo of my ability to create and use local storage and response to user-input with event listeners, not so much for looks or to actually be used, but I might come back and improve it.

Start Button will start a timer that will display game over when there is no time remaining (reaches 0). Incorrect answer will take an additional 5 seconds off of the countdown with the next tick and display "Incorrect" in red. Correct answer will display "Correct" in green. After answering the questions buttons disapear and the "next" button appears, clicking that will pull up the next question. When the timer reaches 0 questions and buttons are hidden. Score is updated and can be saved with initials, most recent score will be displayed at the bottom upon clicking "save score" and will be updated upon refresh or closing the tab and re-opening it.
