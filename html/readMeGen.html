Read-Me-Generator

Back to home page.

This application shows my ability to code in JavaScript and use Node.js npm packages that I am not familiar with, which in this case was inquirer.

This application was made to run in the terminal to make ReadMe.md files quickly. Above you will see a video demonstrating the application.
When you run the program it will prompt you with questions in the terminal/shell where you ran it and then puts your responses in a formatted README.md file.
